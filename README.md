# PCB Ingressi (Stakx)
Input PCB for high power speakers (30A capable) including:
- 4 pole speakON parallel IN (x2)
- 4 pole Phoenix connector OUT
- 4 pole JST connector OUT

## Connectors
[Neutrik NL4MP-M3](https://www.neutrik.com/en/product/nl4mp-m3)
